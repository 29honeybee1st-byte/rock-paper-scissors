## 🖥️ What This App Does

Rock Paper Scissors is a browser game you can open on a Windows PC and play right away. It gives you a fast match against the computer with sound effects, animated backgrounds, and a clean layout that works on different screen sizes.

You choose Rock, Paper, or Scissors, and the game shows the result at once. It is built with HTML, CSS, and JavaScript, so it runs in a web browser and does not need a complicated setup.

## 🎯 How to Play

1. Start the game in your browser
2. Click **Rock**, **Paper**, or **Scissors**
3. The computer makes its choice
4. The game shows who won the round
5. Keep playing to test your luck and timing

The rules are simple:

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## 🛠️ Basic Troubleshooting

If the game does not open:

1. Make sure you extracted the ZIP file first
2. Check that you opened `index.html`
3. Try a different browser
4. Refresh the page
5. Restart the browser

If the layout looks odd:

- Maximize the browser window
- Zoom back to 100%
- Use a current version of your browser

If sound does not work:

- Reload the page
- Unmute the tab
- Check your system audio settings
